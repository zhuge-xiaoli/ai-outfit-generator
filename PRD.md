# AI穿搭提示词生成工具 - 产品需求文档

> 版本：v1.0 | 创建日期：2026-03-17

---

## 一、项目概述

### 1.1 产品定位

一款面向AI穿搭账号运营者的效率工具，通过上传服装/场景图片，自动生成可用于即梦等AI生图工具的提示词，并配套生成小红书爆款文案。

### 1.2 核心价值

| 痛点 | 解决方案 |
|------|----------|
| 写提示词费时费力 | LLM分析图片生成提示词 |
| 文案不知道怎么写 | 自动生成小红书风格标题+正文 |
| 流程分散效率低 | 一次性完成"图→提示词→文案"全流程 |

### 1.3 目标用户

- AI穿搭账号运营者
- 小红书穿搭博主
- 电商服装卖家

## 二、功能清单

### 2.1 MVP（必须完成）

| 功能 | 描述 | 优先级 |
|------|------|--------|
| **图片上传** | 支持上传服装图/场景图，支持拖拽和点击 | P0 |
| **图片分析** | LLM识别图片中的服装特征、颜色、款式、场景元素 | P0 |
| **提示词生成** | 根据分析结果生成即梦提示词（含动作模板） | P0 |
| **文案生成** | 自动生成小红书风格标题+正文 | P0 |
| **结果复制** | 一键复制提示词和文案到剪贴板 | P0 |

### 2.2 扩展功能（后期迭代）

| 功能 | 描述 | 优先级 |
|------|------|--------|
| **模特选择** | 提供预设的好看模特脸提示词供选择 | P1 |
| **动作选择** | 选择预设动作模板（包带互动、慵懒半靠墙等） | P2 |
| **多图批量处理** | 一次上传多张图片，批量生成 | P2 |
| **历史记录** | 保存生成历史 | P2 |

## 三、用户流程

```
┌─────────────────────────────────────────────────────────────┐
│                        用户操作流程                          │
├─────────────────────────────────────────────────────────────┤
│                                                              │
│   1. 上传图片（服装/场景，可多张）                          │
│      ↓                                                      │
│   2. 选择动作模板（可选，默认随机）                          │
│      ↓                                                      │
│   3. 点击「生成」                                            │
│      ↓                                                      │
│   4. 查看结果                                               │
│      ├── 复制提示词                                         │
│      └── 复制文案                                           │
│                                                              │
└─────────────────────────────────────────────────────────────┘
```

## 六、验收标准

- [ ] 可以上传图片并预览
- [ ] LLM能正确分析图片中的服装和场景元素
- [ ] 生成的提示词符合即梦格式要求
- [ ] 生成的小红书文案符合平台风格
- [ ] 可以一键复制结果
